  - Explorar dados para identificar padrões e insights

  - Comunicar resultados para equipes técnicas e de negócio

**CORE SKILLS (essenciais):**

  - Programação em Python

  - Estatística e Probabilidade

  - Machine Learning (supervisionado e não supervisionado)

**NICE-TO-HAVE (complementares):**

  - Engenharia de dados (ETL, pipelines)

  - Visualização de dados e storytelling

**FERRAMENTAS E TECNOLOGIAS:**

  - Pandas, NumPy, Scikit-learn

  - TensorFlow ou PyTorch

  - SQL e bancos de dados

  - Revisão de conceitos e prática de perguntas

## 🚀 PROJETO DE PORTFÓLIO

**PROJETO:** Previsão de Demanda em Setor Crítico

**O QUE FAZER:**  
Construir um modelo que prevê demanda de energia ou ocupação hospitalar usando dados públicos.

**ENTREGÁVEIS:**

  - Notebook com análise exploratória

  - Modelo treinado e validado

  - Dashboard simples com resultados

**CRITÉRIOS DE ACEITAÇÃO:**

  - Dados limpos e bem documentados

  - Modelo com métricas claras (RMSE, accuracy)

  - Repositório organizado no GitHub

**DICA:** Use dados abertos (energia, saúde, transporte) para dar relevância ao projeto.
